Some tweaks I suggest (jailbreak required):

NeonBoard

A modern WinterBoard replacement (iOS 7+).

Get it from ArtikusHG's repo

Solid Glass

A Liquid Glass theme for ancient devices (SnowBoard/Anemone/NeonBoard) needed.
Both iOS 26 (Solid Glass 2) and iOS 27 (Solid Glass 27) available.

Get it from iVintageLover repo
Solid Glass 27 or Solid Glass 2

Zeppelin

A nice mod that lets you customize your carrier name and even add an icon!

Get it from ModMyi Archive

Eclipse

Add dark mode to iOS 7-10. Note you need to download the correct version for your iOS version.

Get it from the BigBoss repo

AppSync Unified

Install unsigned IPAs forever!

Get it from LukeZGD repo

TubeReplacer

An alternative to TubeRepair which is client-side rather than server-side. This means login still works.
In case you don't know, TubeRepair makes ancient YouTube versions still work today.
Requires YouTube 1.0.0-1.4.0.

Get it from Preloading

Filza

A modern, freemium alternative to iFile as a root file explorer.

Get it from Tigi Software or BigBoss
